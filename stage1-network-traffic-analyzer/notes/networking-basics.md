# Networking Basics

## What is a Computer Network?

A computer network is a collection of interconnected devices that communicate with each other to exchange data and share resources.


## Why Do We Need Networking?

- Communication
- Resource Sharing
- Internet Access
- File Transfer
- Remote Access


## Types of Networks

- LAN (Local Area Network)
- MAN (Metropolitan Area Network)
- WAN (Wide Area Network)


## Real-world Example

A banking system consists of multiple branches connected over a secure WAN. Every ATM, employee workstation, database server, and customer-facing application communicates through this network.
